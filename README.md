# AI-Driven YouTube Content Creation – Prompt Chain

This repository documents a structured **prompt engineering workflow** used to create a complete YouTube video (script, video, and thumbnail) using multiple AI tools.

The project demonstrates how prompt chaining, iteration, and tool selection can be used to guide AI systems toward high-quality, real-world content creation.

---

## 🎯 Project Overview

**Video Topic:**  
*I Asked ChatGPT to Run My Day – Here’s What Happened*

**Genre:** Productivity / Lifestyle

**Objective:**  
To design and document a clear, step-by-step prompt chain that leverages AI tools intentionally to:
- Analyze audience psychology
- Generate high-performing YouTube titles
- Write and refine a retention-focused script
- Create AI-assisted videos
- Design scroll-stopping thumbnails

---

## 🧠 Prompt Engineering Approach

This project follows a **prompt chain methodology**, where:
- Each prompt has a defined role
- Outputs from one step guide the next
- Prompts are refined based on output quality
- Tools are selected based on their strengths

This mirrors a real-world AI-assisted creator workflow.

## 🔗 Prompt Chain Breakdown

### 1️⃣ Audience & Narrative Analysis
Defines the target audience, emotional hook, differentiation, and core message.

### 2️⃣ Title Generation
Produces curiosity-driven, non-clickbait YouTube titles optimized for CTR.

### 3️⃣ Script Generation
Creates a story-driven script structured for retention and AI video tools.

### 4️⃣ Script Refinement
Improves pacing, transitions, and emotional depth without rewriting.

### 5️⃣ Thumbnail Prompting
Designs emotionally expressive thumbnail concepts and AI image prompts.

---

## 🔁 Iteration Insight

Initial thumbnail generations appeared visually generic.  
By refining prompts to emphasize **emotional tension and internal conflict**, the visual quality and relevance of outputs improved significantly.

This highlights the importance of **prompt specificity and iteration** in prompt engineering.

---

## 🧠 Key Learnings

- Prompt engineering is about **directing intent**, not just asking questions
- Iteration is essential for improving AI output quality
- Emotional framing is as important as technical clarity
- Proper documentation is critical in AI workflows
